RAM-computer's short-term memory

RAM (Random Access Memory) is a type of computer memory that allows data to be stored and retrieved quickly by the CPU. It is volatile, meaning it loses its contents when the power is turned off. RAM is essential for running programs and multitasking efficiently on a computer.The data which is easily modifiable are generally stored in the RAM. RAM is of two types:

1.SRAM
2.DRAM

Static Random Access Memory (SRAM)


    Characteristics of Static RAM

  • Static RAM is much faster than DRAM.
  • Static RAM has greater storage than DRAM.
  • Static RAM takes less power to perform.

    Advantages of Static RAM

  • Static RAM has low power consumption.
  • Static RAM has faster access speeds than DRAM.
  • Static RAM helps in creating a speed-sensitive cache.

    Disadvantages of Static RAM

  • Static RAM has less memory capacity.
  • SRAM has high costs of manufacturing than DRAM.
  • Static Ram comprises of more complex design.

Dynamic Random Access Memory (DRAM)


    Characteristics of Dynamic RAM

  • Dynamic RAM is slower in comparison to SRAM.
  • Dynamic RAM is less costly than SRAM.
  • Dynamic RAM has high power consumption.

    Advantages of Dynamic RAM

  • Dynamic RAM has Low costs of manufacturing than SRAM.
  • Dynamic RAM has greater memory capacities.
  • Dynamic RAM does not need to refresh its memory contents.

    Disadvantages of Dynamic RAM

  • Dynamic RAM has a slow access speed.
  • Dynamic RAM has high power consumption.
  • Dynamic RAM has high power consumption.

SRAM V/S DRAM

Aspect SDRAM DRAM
Speed Operates synchronously with system clock Operates asynchronously
Access Time Lower access time due to synchronization Higher access time compared to SDRAM
Refresh Rate Does not require as frequent refreshing Requires periodic refreshing
Cost Generally more expensive than DRAM Less expensive compared to SDRAM
Usage Commonly used in modern systems Historically widely used, but less common in modern systems
